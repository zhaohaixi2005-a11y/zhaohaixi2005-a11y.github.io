Project Gateway 04

Profiling Hypoxia Responses in Colonic Organoids

An organoid and single-cell analysis project centered on hypoxia-response mapping, microfluidic perturbation, and interpretable cell-state profiling.

Oct 2024 - Mar 2025 Research Project

Academic Guidance

Institution HUST
Supervisor
Prof. Yiwei Li
Microfluidics Single-cell RNA-seq PCA/UMAP Cell State Annotation

Project Overview

Investigated how hypoxia and nutrient deprivation alter cell-state composition in colonic organoids using a microfluidic pathological-environment platform.

Research Motivation

Better mechanistic maps of stress-adapted cellular responses can improve understanding of microenvironment-driven phenotypic shifts.

Methods / Pipeline

  • Adjusted PCA/UMAP offset settings to improve cluster resolution.
  • Modified mitochondrial-content QC threshold to 30% for hypoxia signal sensitivity.
  • Optimized preprocessing pipeline for clearer subpopulation annotation.
  • Quantified DEGs and condition-specific expression shifts under chip constraints.

My Contributions

  • Refined key dimensionality reduction and QC parameters to recover biologically relevant signals.
  • Improved interpretability of downstream bubble-plot and annotation outputs.
  • Helped link microfluidic perturbation design with transcriptomic readout strategy.

Key Outcomes

  • Enhanced resolution of hypoxia-associated cellular subpopulations.
  • Improved quality of integrated scRNA-seq interpretation pipeline.
  • Generated a structured analysis pattern for future organoid perturbation studies.

Related Tools / Skills

  • Scanpy, single-cell QC/annotation, dimensionality reduction diagnostics
  • Microfluidic experimental context integration
  • Statistical interpretation of condition-specific transcriptomic changes

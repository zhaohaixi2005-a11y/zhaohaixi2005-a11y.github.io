Project Overview

This project aims to localize and characterize lysine-linked PTM states by designing de novo binders that enrich modified species with structural specificity.

Research Motivation

Existing PTM capture approaches often struggle with selectivity and spatial precision. A topology-constrained binder architecture can improve recognition fidelity.

Methods / Pipeline

  • Designed pure-helical pseudocycle backbones to enforce geometric constraints.
  • Integrated BoltzGen + RFdiffusion + LigandMPNN for generation and sequence design.
  • Applied geometry-aware filtering for helix packing, stability, and binding pocket quality.
  • Prioritized candidates using a fully computational PTM-specific screening workflow.

My Contributions

  • Constructed end-to-end computational workflow for PTM-focused binder generation and triage.
  • Defined structure-quality heuristics and thresholds for candidate ranking.
  • Built an iterative analysis loop linking topology design and docking-aware refinement.

Key Outcomes

  • Generated constrained binder families with improved structural coherence.
  • Reduced downstream search space through geometry-first candidate filtering.
  • Established reusable design protocol for future PTM classes.

Related Tools / Skills

  • BoltzGen, RFdiffusion, LigandMPNN, structural scoring scripts
  • Protein geometry evaluation and quality control workflow design
  • Computational prioritization for experimental handoff
